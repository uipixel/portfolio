## О проекте

Портфолио оформлено, как командная строка: посетитель вводит команды
(или нажимает кнопки) и получает информацию обо мне и моих работах —
реализованных проектах и дизайн-концептах. Внутри спрятана и мини-игра.

## Команды

| Команда    | Что показывает                  |
|------------|---------------------------------|
| `about`    | обо мне                         |
| `skills`   | инструменты и навыки            |
| `projects` | реализованные проекты           |
| `concepts` | дизайн-концепты                 |
| `game`     | мини-игра (космо-шутер)         |
